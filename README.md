# Pokemon Trainer Finance Manager

## Introduction
This project manages financial transactions for your average Pokemon Trainer by categorizing them into commonly found 
income and expenses.

## Features
- Categorized financial tracking with customizable descriptions
- Easy-to-use interface to input new transactions
- Ability to save and load date from previous sessions
- Summary of personal finances with visual representation

## How To Run
1. The termcharts module needs to be installed with `python -m pip install termcharts` first
2. Run `main.py` to start the program

## Usage
- Create a `Transaction` object for each transaction.
- Assign transactions to categories.
- Filter transactions according to category
- Search for transactions through description 
- Generate summaries and bar graphs to analyze financial data.
